## При верстке данного макета, использовалась система сборки nodejs, angular

### Для установки всех зависимостей

    npm i

Убедитесь, что на вашем компьютере установлен npm

### Для запуска сборщика

    npm run dev

### Используемые средства

    - angular
    - mongoose
    - nodejs
    - express
    - passport.js
    - chart.js
    - materialize.css
